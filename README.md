# ML Powered Social Media Security Detection and Blocking System

## Overview

This project is a secure social media platform developed using Java, JSP, Servlets, and MySQL. The system detects spam and harmful content using NLP preprocessing, Bloom Filter techniques, and automated moderation mechanisms.

## Features

* User Registration and Login
* Friend Request System
* Real-Time Chat
* Post and Comment Management
* Spam Detection and Filtering
* Automatic IP Blocking
* Admin Monitoring Dashboard
* Event Management System

## Technologies Used

* Java
* JSP & Servlets
* MySQL
* Apache Tomcat
* HTML/CSS
* Apache Commons FileUpload

## Project Modules

1. User Management
2. Spam Detection System
3. Chat and Communication
4. Admin Dashboard
5. Event Management

## Future Enhancements

* BERT/GPT-based moderation
* Advanced NLP models
* CAPTCHA and 2FA
* Blockchain-based moderation

## How to Run

1. Import project into NetBeans
2. Configure MySQL database
3. Add required JAR files
4. Deploy on Apache Tomcat
5. Run the project
